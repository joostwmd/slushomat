## Description

Add an **admin-only** nested router (e.g. `admin.templateProduct` under `adminRouter`) for **template products** that provides:

| Concern | Behavior |
|---------|----------|
| **CRUD** | List (include image metadata via join or secondary query), create, update, delete **template products** (`template_product`) |
| **Upload handshake** | `requestImageUpload` — input: `templateProductId`, `contentType`, optional `filename` (extension only); output: `path`, `token`, `bucket` (names may vary) |
| **Confirm image** | `confirmTemplateProductImage` (name flexible) — input: `templateProductId` + `path` that **must match** server-issued prefix; upsert `template_product_image` |
| **Delete** | Remove Storage object(s) via `SupabaseStorageService` **then** delete DB rows (or document strict ordering if not transactional with Storage) |

All procedures use **`adminProcedure`** from [`apps/server/src/trpc/procedures.ts`](apps/server/src/trpc/procedures.ts). Use **Zod** for inputs/outputs; map errors to **`TRPCError`** (`NOT_FOUND`, `BAD_REQUEST`, `FORBIDDEN` as appropriate).

**Path safety:** Server generates **opaque** object paths (e.g. `templates/{templateProductId}/{uuid}.ext`). **Never** trust client-supplied full paths. Validate **`templateProductId`** exists before minting upload URL.

**Storage service wiring:** Prefer a **small factory** in the router module (or dedicated `lib/`) — avoid bloating global `createContext` unless the project already centralizes services.

Register the router in [`apps/server/src/trpc/routers/admin.ts`](apps/server/src/trpc/routers/admin.ts). Root [`apps/server/src/trpc/router.ts`](apps/server/src/trpc/router.ts) stays **`admin: adminRouter`** only.

### `adminProcedure` (auth gate)

```32:51:apps/server/src/trpc/procedures.ts
export const adminProcedure = withErrorMapper.use(({ ctx, next }) => {
  if (!ctx.session?.user) {
    throw new TRPCError({
      code: "UNAUTHORIZED",
      message: "Authentication required",
    });
  }
  if (!isAdmin(ctx)) {
    throw new TRPCError({
      code: "FORBIDDEN",
      message: "Admin role required",
    });
  }
  return next({
    ctx: {
      ...ctx,
      user: ctx.session.user,
      session: ctx.session,
    },
  });
});
```
