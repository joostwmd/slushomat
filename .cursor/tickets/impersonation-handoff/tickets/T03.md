# T03 — Operator Handoff Page

**Task ID:** T03  
**Agent:** frontend-agent  
**Depends on:** T01  
**Commit message:** `feat(operator): add /auth/handoff page for impersonation session handoff`

---

## Description

Implement the handoff page that exchanges the one-time token for a session. Create route `/auth/handoff` outside the protected layout. Read `token` from query params. If missing → redirect to `/sign-in`. If present → call `authClient.oneTimeToken.verify({ token })`. On success → redirect to `/dashboard`. On error → show "This link has expired or is invalid" with "Go to sign in" button.

## Implementation Reference

### Route Structure

Operator uses TanStack Router. Create `apps/operator-frontend/src/routes/auth/handoff.tsx`. Ensure `auth` is a path (not layout) so route is `/auth/handoff`. Create `auth/` directory with `handoff.tsx` if it does not exist.

Reference: `apps/operator-frontend/src/routes/sign-in.tsx` — flat route at root. For `/auth/handoff`, use `routes/auth/handoff.tsx` — TanStack Router infers path `/auth/handoff`.

### Page Logic

```typescript
function HandoffPage() {
  const [state, setState] = useState<"loading" | "error" | "redirecting">("loading");
  const navigate = useNavigate();
  const { token } = Route.useSearch();

  useEffect(() => {
    if (!token) {
      navigate({ to: "/sign-in" });
      return;
    }
    authClient.oneTimeToken
      .verify({ token })
      .then(() => {
        setState("redirecting");
        navigate({ to: "/dashboard" });
      })
      .catch(() => {
        setState("error");
      });
  }, [token, navigate]);

  if (!token) return null;

  if (state === "loading" || state === "redirecting") {
    return (
      <div className="flex flex-col items-center justify-center gap-2 min-h-[calc(100svh-theme(spacing.14))]">
        <Loader2 className="h-6 w-6 animate-spin" />
        <p>Signing in...</p>
      </div>
    );
  }

  return (
    <div className="flex min-h-[calc(100svh-theme(spacing.14))] flex-col items-center justify-center gap-4 px-4">
      <div role="alert" className="text-center space-y-2">
        <p className="text-destructive">This link has expired or is invalid.</p>
        <Button asChild>
          <Link to="/sign-in">Go to sign in</Link>
        </Button>
      </div>
    </div>
  );
}
```

### Route Config

```typescript
export const Route = createFileRoute("/auth/handoff")({
  component: HandoffPage,
  validateSearch: (search) => ({ token: search.token as string | undefined }),
});
```
