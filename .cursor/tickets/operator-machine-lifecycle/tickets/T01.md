# T01 — DB schema: entities, versioned contracts, deployments, slots

**Task ID:** T01  
**Agent:** db-agent  
**Depends on:** T00  
**Parallel with:** —

## Description

| Area | Implementation |
|------|----------------|
| **`business_entity`** | `defineStaticEntity` from **`@slushomat/db/model-factory`** (or `../model-factory` from schema files): **scope `org`**, **`softDelete: true`**, columns per [00-requirements.md](../00-requirements.md): `name`, `legalName`, `legalForm`, `vatId`, address, `country` default **DE**, timestamps. |
| **`operator_contract`** | `defineVersionedEntity`: **scope `org`**. **`baseColumns`:** `businessEntityId` → `business_entity`, `machineId` → `machine`. **`versionColumns`:** `status` (`draft` \| `active` \| `terminated`), `effectiveDate`, `endedAt` (nullable), `monthlyRentInCents`, `revenueShareBasisPoints` (0–10000), PDF storage (bucket + object path or equivalent — support signed URLs in T03), `notes` (nullable). |
| **Changes table** | From factory (`operator_contract_change` or generated name) with actor/action audit per skill. |
| **`machine_slot`** | **Plain Drizzle** `pgEnum`: `left`, `middle`, `right`. |
| **`machine_deployment`** | **Plain Drizzle:** `machineId`, `businessEntityId`, `startedAt`, `endedAt` (null = open). **Partial unique:** one row per `machineId` WHERE `endedAt IS NULL`. |
| **`machine_slot_config`** | **Plain Drizzle:** `machineDeploymentId`, `slot`, nullable `operatorProductId` → **`operator_product.id`**. **Unique** `(machineDeploymentId, slot)`. |
| **Indexes** | Admin filters: `organizationId`, `machineId`, paths for contract status via current version / joins. |
| **Exports** | [`packages/db/src/schema/index.ts`](../../../packages/db/src/schema/index.ts); merge factory `*Relations` with manual relations for deployments/slots. |
| **Migration** | `drizzle-kit generate` + apply; [03-plan.md](../03-plan.md) §0 — backup DB; confirm `operator_product` exists. |

**Invariant:** At most **one active** contract per `machineId` when activating — enforce in T02/T03 transaction; optional DB constraint if feasible.
