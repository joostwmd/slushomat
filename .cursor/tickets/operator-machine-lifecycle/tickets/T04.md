## Description

| Router domain | Behaviour |
|---------------|-----------|
| **`operator.businessEntity`** | `list`, `create`, `update`, `softDelete` — every call includes **`orgSlug`**; resolve `organizationId` + **`assertUserMemberOfOrg`**. |
| **`operator.operatorContract`** | `list` (optional `machineId`), `get` — **read-only**; **draft, active, terminated**; version history if low-cost. |
| **`operator.machineSlot`** | `getConfigForMachine(orgSlug, machineId)` — open deployment + three slots. `setSlots(...)` — deployment open; each product **null or** `operator_product.organizationId === resolved org`. |
| **Auth** | **`operatorProcedure`** + **per-route** org + membership (do not rely on “any membership”). |

Extend [`operator.ts`](../../../apps/server/src/trpc/routers/operator.ts) (pattern: [`operator-products` router](../../../apps/server/src/trpc/routers/operator-products.ts) — `orgSlug` + `resolveOrgWithMembership`).
