## Description

| Deliverable | Detail |
|-------------|--------|
| **New file** | `packages/db/src/schema/purchase.ts` |
| **Table** | `purchase` — immutable audit records of machine transactions |
| **Enum reuse** | Import `machineSlotEnum` from `./machine-lifecycle` — **do not redefine** |
| **FKs** | `machineId → machine.id (restrict)`, `organizationId → organization.id (restrict)`, `businessEntityId → business_entity.id (set null)`, `operatorProductId → operator_product.id (restrict)` |
| **Nullable** | `businessEntityId` only — null when no open deployment at time of purchase |
| **Indexes** | `(organizationId, purchasedAt DESC)`, `(machineId, purchasedAt DESC)`, `(businessEntityId, purchasedAt DESC)` |
| **Export** | Add `export * from "./purchase"` to `packages/db/src/schema/index.ts` |
| **Migration** | Run `drizzle-kit generate` to produce migration file |

### Column specification

```ts
{
  id:                text("id").primaryKey(),
  machineId:         text("machine_id").notNull().references(() => machine.id, { onDelete: "restrict" }),
  organizationId:    text("organization_id").notNull().references(() => organization.id, { onDelete: "restrict" }),
  businessEntityId:  text("business_entity_id").references(() => businessEntity.id, { onDelete: "set null" }),
  operatorProductId: text("operator_product_id").notNull().references(() => operatorProduct.id, { onDelete: "restrict" }),
  slot:              machineSlotEnum("slot").notNull(),
  amountInCents:     integer("amount_in_cents").notNull(),
  purchasedAt:       timestamp("purchased_at").defaultNow().notNull(),
  createdAt:         timestamp("created_at").defaultNow().notNull(),
}
```
