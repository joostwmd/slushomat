### `PurchaseRow` type (define in shared types or inline)

```ts
export interface PurchaseRow {
  id: string;
  purchasedAt: Date;
  machineId: string;
  slot: "left" | "middle" | "right";
  productName: string;
  amountInCents: number;
  businessEntityName?: string | null;
  businessEntityId?: string | null;
}
```

### `PurchasesTable` props

```ts
interface PurchasesTableProps {
  data: PurchaseRow[];
  isLoading: boolean;
  filters?: {
    dateFrom?: Date | null;
    dateTo?: Date | null;
    businessEntityId?: string | null;
    machineId?: string | null;
  };
  filterOptions?: {
    businessEntities?: { id: string; name: string }[];
    machines?: { id: string }[];
  };
  onFiltersChange?: (filters: PurchasesTableProps["filters"]) => void;
  onExportCsv?: () => void;
  showMachineColumn?: boolean;   // default true
  showEntityColumn?: boolean;    // default true
}
```

### Column definitions

| Column | Key | Format |
|--------|-----|--------|
| Date / Time | `purchasedAt` | `dd.MM.yyyy HH:mm` |
| Machine | `machineId` | Monospace, truncated; hidden when `!showMachineColumn` |
| Slot | `slot` | Capitalised (`Left` / `Middle` / `Right`) |
| Product | `productName` | String |
| Amount | `amountInCents` | `€ X,XX` (EUR, 2 decimal places) |
| Business Entity | `businessEntityName` | String or `—`; hidden when `!showEntityColumn` |

### Filter row (above table)

- "From" date picker input
- "To" date picker input
- "Entity" select (from `filterOptions.businessEntities`; "All entities" default)
- "Machine" select (from `filterOptions.machines`; "All machines" default; shown when `showMachineColumn`)
- "Export CSV" button with download icon — shown when `onExportCsv` is provided

### States

- Loading: 5-row skeleton matching column widths.
- Empty (no filter): "No purchases yet" with an icon.
- Empty (filter active): "No results for this filter."

### `exportPurchasesToZip` utility

```ts
export async function exportPurchasesToZip(
  rows: PurchaseRow[],
  filenamePrefix: string   // e.g. "purchases-myorg" or "purchases-machine-m-1"
): Promise<void>
```

**Implementation:**
1. Build CSV string:
   - Header: `Date,Time,Machine ID,Slot,Product,Amount (EUR),Business Entity`
   - Each row: `YYYY-MM-DD,HH:MM:SS,<machineId>,<slot>,<productName>,<X.XX>,<entityName|"">`
   - Amount: `(amountInCents / 100).toFixed(2)`
   - Date/Time: UTC or local — document the choice in a comment.
2. Use `jszip` (check if installed in `packages/ui/package.json`; add if missing):
   ```ts
   const zip = new JSZip();
   zip.file("purchases.csv", csvString);
   const blob = await zip.generateAsync({ type: "blob" });
   ```
3. Trigger download:
   ```ts
   const url = URL.createObjectURL(blob);
   const a = document.createElement("a");
   a.href = url;
   a.download = `${filenamePrefix}-${format(new Date(), "yyyy-MM-dd")}.zip`;
   a.click();
   URL.revokeObjectURL(url);
   ```
