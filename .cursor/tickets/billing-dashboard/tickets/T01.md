## Description

| Deliverable | Detail |
|-------------|--------|
| **Route file** | `apps/machine-server/src/routes/purchase.ts` |
| **Auth** | Protected by existing `machineAuthMiddleware` (sets `c.get("machineId")`) |
| **HTTP** | `POST /purchase` → `201` on success, `422` if no active contract |
| **Org resolution** | Query `operator_contract` base joined with its current version; filter `machineId + status = 'active'` |
| **Entity resolution** | Query `machine_deployment` where `machineId = :id AND endedAt IS NULL`; nullable |
| **Error code** | Add `NO_ACTIVE_CONTRACT` to `apps/machine-server/src/errors.ts` |
| **Wire** | Register route + middleware in `apps/machine-server/src/index.ts` |

### Input schema (Zod)

```ts
z.object({
  operatorProductId: z.string().min(1),
  slot: z.enum(["left", "middle", "right"]),
  amountInCents: z.number().int().positive(),
})
```

### Resolution logic

```
machineId       ← c.get("machineId")

organizationId  ← SELECT oc.organization_id
                  FROM operator_contract oc
                  JOIN operator_contract_version ocv ON ocv.id = oc.current_version_id
                  WHERE oc.machine_id = :machineId AND ocv.status = 'active'
                  LIMIT 1
                → if not found → 422 { code: "NO_ACTIVE_CONTRACT" }

businessEntityId ← SELECT md.business_entity_id
                   FROM machine_deployment md
                   WHERE md.machine_id = :machineId AND md.ended_at IS NULL
                   LIMIT 1
                 → null if not found (allowed)
```

### Success response

```json
{ "id": "uuid", "purchasedAt": "ISO-8601 timestamp" }
```
