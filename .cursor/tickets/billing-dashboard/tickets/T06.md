## Description

---

### 1. `/_protected/$orgSlug/purchases` (new route)

**File:** `apps/operator-frontend/src/routes/_protected/$orgSlug/purchases.tsx`

```ts
export const Route = createFileRoute("/_protected/$orgSlug/purchases")({
  component: OperatorPurchasesPage,
});
```

**Filter state** — manage in component state (or URL search params for shareability — implementation choice; prefer URL search params for better UX):

```ts
const [filters, setFilters] = useState({
  dateFrom: null,
  dateTo: null,
  businessEntityId: null,
  machineId: null,
});
```

**Data fetching:**

```ts
const { orgSlug } = Route.useParams();

// Purchases
const purchasesQuery = useQuery(
  trpc.operator.purchase.list.queryOptions({
    orgSlug,
    machineId: filters.machineId ?? undefined,
    businessEntityId: filters.businessEntityId ?? undefined,
    startDate: filters.dateFrom ?? undefined,
    endDate: filters.dateTo ?? undefined,
  }),
);

// Machines for org (for machine cards + machine filter dropdown)
const machinesQuery = useQuery(trpc.operator.machine.list.queryOptions({ orgSlug }));

// Business entities for filter dropdown
const entitiesQuery = useQuery(trpc.operator.businessEntity.list.queryOptions({ orgSlug }));
```

**Page layout:**

```
Purchases                                [Export CSV ↓]

[Date From] [Date To]  [Entity ▾]  [Machine ▾]

<PurchasesTable
  data={purchasesQuery.data ?? []}
  isLoading={purchasesQuery.isPending}
  filters={filters}
  filterOptions={{
    businessEntities: entitiesQuery.data,
    machines: machinesQuery.data,
  }}
  onFiltersChange={setFilters}
  onExportCsv={handleExport}
  showMachineColumn={true}
  showEntityColumn={true}
/>

--- Your Machines ---

<Grid of machine cards>
```

**Export handler:**

```ts
async function handleExport() {
  // If data is paginated and not fully loaded, fetch all with same filters (no limit)
  const rows = purchasesQuery.data ?? [];
  await exportPurchasesToZip(rows, `purchases-${orgSlug}`);
}
```

**Machine cards grid:**

```ts
{machines.map((m) => (
  <Link to="/$orgSlug/machines/$machineId/purchases" params={{ orgSlug, machineId: m.id }}>
    <Card>
      <CardTitle>{m.id}</CardTitle>
      <CardDescription>Model {m.versionNumber}</CardDescription>
    </Card>
  </Link>
))}
```

Empty state if no machines: "No machines under contract yet."

### 2. `/_protected/$orgSlug/machines.$machineId.purchases` (new route)

**File:** `apps/operator-frontend/src/routes/_protected/$orgSlug/machines.$machineId.purchases.tsx`

```ts
export const Route = createFileRoute(
  "/_protected/$orgSlug/machines/$machineId/purchases",
)({ component: MachinePurchasesPage });
```

**Layout:**

```
← Machines   Machine [machineId] — Purchases      [Export CSV ↓]

[Date From] [Date To]  [Entity ▾]

<PurchasesTable
  data={purchasesQuery.data ?? []}
  isLoading={purchasesQuery.isPending}
  filters={dateFilters}
  filterOptions={{ businessEntities: entitiesQuery.data }}
  onFiltersChange={setDateFilters}
  onExportCsv={handleExport}
  showMachineColumn={false}
  showEntityColumn={true}
/>
```

**Data fetching:**

```ts
const { orgSlug, machineId } = Route.useParams();
const purchasesQuery = useQuery(
  trpc.operator.purchase.list.queryOptions({
    orgSlug,
    machineId,
    ...dateFilters,
  }),
);
```

**Export filename:** `purchases-${machineId}-${format(new Date(), "yyyy-MM-dd")}.zip`
