# T03 — Create Customer Wizard UI

**Task ID:** T03  
**Agent:** frontend-agent  
**Depends on:** T02  
**Commit message:** `feat(admin): add create customer wizard`

---

## Description

Build the two-step create-customer wizard per `01-ui-spec.md`. Step 1: form with email, password, name; submit via `authClient.admin.createUser`. Step 2: form with name, slug, optional logo and metadata; submit via `trpc.admin.createOrganization.mutate()`. Display user summary (read-only) in step 2. Success state: "Customer created: {name} / {org name}" with "Go to dashboard" and "Create another customer" actions. Add entry link from admin dashboard.

**Route:** `/create-customer` under `_admin` layout (TanStack Router: `apps/admin-frontend/src/routes/_admin/create-customer.tsx`)

## Implementation Reference

### Skills: `skills/frontend/shadcn.md`, `skills/frontend/design/harden.md`

**Shadcn components:** Card, CardHeader, CardTitle, CardDescription, CardContent, CardFooter, Input, Label, Button, (optional: Textarea for metadata)

**State management:**
- `step`: 1 | 2
- `user`: { id, email, name } | null (from step 1 response)
- `loading`: boolean
- `error`: string | null
- `success`: { userName, orgName } | null

**Step 1 submit:**
```ts
const handleStep1 = async (e) => {
  e.preventDefault();
  setError(null);
  setLoading(true);
  try {
    const res = await authClient.admin.createUser({ email, password, name, role: "user" });
    setUser({ id: res.id, email, name });
    setStep(2);
  } catch (err) {
    setError(err.message ?? "Failed to create user");
  } finally {
    setLoading(false);
  }
};
```

**Step 2 submit:**
```ts
const handleStep2 = async (e) => {
  e.preventDefault();
  if (!user) return;
  setError(null);
  setLoading(true);
  try {
    const res = await trpc.admin.createOrganization.mutate({
      name: orgName,
      slug: orgSlug,
      logo: orgLogo || undefined,
      metadata: orgMetadata ? JSON.parse(orgMetadata) : undefined,
      userId: user.id,
    });
    setSuccess({ userName: user.name, orgName: res.name });
  } catch (err) {
    setError(err.message ?? "Failed to create organization");
  } finally {
    setLoading(false);
  }
};
```

**Create another:** Reset `step`, `user`, `success`, clear form fields.
