## Description

Optional **review pass** after T05 and T06 ship. No code changes required unless findings are filed as follow-up tickets.

Focus:

1. **Org isolation** — Every `operator.product.*` procedure resolves org from **`orgSlug`**, asserts **`member(userId, organizationId)`**, and scopes reads/writes to `operator_product.organization_id`. Attempts to pass another org’s slug without membership must fail (`FORBIDDEN` / `NOT_FOUND` as appropriate).
2. **Shared image safety** — Copy-from-template shares `product_image_id`. Deleting one operator product must **not** remove Storage if a template or another product still references the same `product_image` row (verify `deleteProductImageIfUnreferenced` usage on all detach/delete paths).
3. **Admin regression** — Template products: list with signed image URLs, CRUD, replace image, delete product, no orphan UI errors.
4. **a11y / UX** — Empty state: heading/description hierarchy, focus order into actions, sheet/modal focus trap, form labels on composite fields, error text associated with inputs.
